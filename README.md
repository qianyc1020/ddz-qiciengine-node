# qiciengine + node.js 斗地主

## 1）客户端 支持联网和ai
*  客户端是用qiciengine导入客户端工程然后打开，而不是cocos creator，qiciengine是一款类似于unity的开源编辑器，很强大。

## 2）服务器
*  服务器是node.js 6.11.1，越来越喜欢这个玩意了,别的node版本报错
